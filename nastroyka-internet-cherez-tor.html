Настройка интернет через Tor

В настоящий момент, для использования сети Tor в РБ, необходимо настроить подключение через так называемые "мосты" Tor сети:

1. Скачать Tor Browser:  Windows   Mac   Linux   Android

Инструкция по настройке Windows версии - ниже. Mac и Linux настраиваются похожим образом. Настройки Android версии - в самом конце.

2. Запустить скачанный файл. Выбрать язык, нажать "Установить"

Нажать "Готово", на рабочем столе появится папка и ярлык для запуска:

3. Запустить Start Tor Browser. Нажать "Настроить":

4. Выбрать "Tor заблокирован в моей стране" -> "Запросить мост у torproject.org".

Нажать "Запрос моста..." :

5. Ввести цифры с картинки, нажать "ОК":

6. Пропишутся мосты. Нажать "Соединиться":

7. Ожидаем подключение к сети и потом пользуемся, как обычным браузером:

Настройка Android версии:

1. Нажать иконку настроек:

2. Включить переключатель "Интернет цензура":

3. Нажать "Выбор моста" -> obfs4

4. Два раза нажать стрелку "назад", вернуться на первый экран. Нажать "СОЕДИНЕНИЕ":

5. Если не подключается - пробуем выбирать meek-azure в пункте 3.
